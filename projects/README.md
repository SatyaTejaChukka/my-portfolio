# Project Screenshots

Add your project screenshots to this folder with the following filenames:

## Required Images

1. `stroke-prediction.png` - Stroke Prediction App screenshot
2. `house-price.png` - House Price Prediction screenshot
3. `traffic-accident.png` - Traffic Accident Prediction screenshot
4. `namaste-icd.png` - Namaste Codes to ICD-11 Mapping screenshot
5. `bloch-sphere.png` - Bloch Sphere Explorer screenshot
6. `interview-master.png` - Interview Master screenshot
7. `money-manage.png` - Money Manage screenshot

## Recommended Specifications

- **Format**: PNG or WebP (WebP preferred for better compression)
- **Dimensions**: 800x450px (16:9 aspect ratio) or similar
- **File Size**: Keep under 200KB per image for fast loading
- **Content**: Show the main interface/dashboard of each project

## Tips for Screenshots

1. Use browser dev tools to set a consistent viewport size
2. Capture the most impressive/representative view of each project
3. Ensure text is readable
4. Use light/dark mode that matches your portfolio theme

## Image Optimization

Before adding images, optimize them using:
- [Squoosh](https://squoosh.app/) - Google's image optimizer
- [TinyPNG](https://tinypng.com/) - PNG/WebP compression
- [ImageOptim](https://imageoptim.com/) - Mac app for batch optimization

The portfolio will gracefully fall back to emoji placeholders if images are missing.
